超级文件批量处理工具 - 专业版 使用说明文档
软件简介
超级文件批量处理工具是一款功能全面的文件管理软件，提供70+种文件处理功能，包括批量重命名、格式转换、图片处理、文件加密等专业功能。

主要特性
批量操作：支持同时处理大量文件

格式支持：支持图片、文档、音频、视频等多种格式

智能预览：实时预览文件内容和处理效果

安全可靠：操作前自动备份，支持撤销/重做

用户友好：直观的图形界面，多语言支持

系统要求
操作系统：Windows 7/8/10/11（64位）

内存：至少 4GB RAM（推荐8GB）

磁盘空间：至少 200MB 可用空间

屏幕分辨率：1366×768 或更高

重命名功能
基本重命名规则
前缀和后缀添加
使用方法：

在"前缀"输入框中输入要添加的文本

在"后缀"输入框中输入要添加的文本

确认无误后点击"应用重命名"

示例：

前缀"IMG_" + 文件"photo.jpg" = "IMG_photo.jpg"

文件"document" + 后缀"_backup" = "document_backup.txt"

文本替换
使用方法：

在"替换"区域的第一个输入框输入要查找的文本

大写：全部转换为大写字母

示例：

"My Document" → "my document"（小写）

"my document" → "MY DOCUMENT"（大写）
